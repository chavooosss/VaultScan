Google ile girişte ne alıyoruz

Sadece Google hesabından gelen ad, e-posta adresi ve profil fotoğrafı bağlantısı. Şifreni hiçbir zaman görmüyoruz — giriş tamamen Google üzerinden yapılır.

AI API key'lerin

VaultScan kendi AI key'ini kullanmaz — Claude/ChatGPT/Gemini için girdiğin key'ler senindir ve veritabanında şifrelenmiş olarak saklanır. Hiçbir ekranda veya API yanıtında düz metin olarak geri gösterilmez. İstediğin zaman Ayarlar sayfasından silebilirsin.

Analiz ettiğin kod ne olur

VaultScan, yapıştırdığın/yüklediğin kodu veya GitHub repo içeriğini kendi sunucusunda saklamaz. Kod, seçtiğin AI sağlayıcısına (Claude/ChatGPT/Gemini) senin kendi API key'inle iletilir ve analiz orada yapılır.

Bu nedenle gerçekten gizli/hassas kod paylaşıyorsan, seçtiğin AI sağlayıcının kendi gizlilik politikasını da göz önünde bulundur — o aşamada veri o sağlayıcının sunucularında işlenir.

Analiz geçmişi

AI'nin ürettiği rapor/bulgular (hangi açıklar bulundu, hangi AI(ler) tespit etti, ne zaman yapıldı) hesabınla ilişkilendirilip Geçmiş sayfasında saklanır — bu sayede eski analizlerine dönüp bakabilirsin. Analiz ettiğin kodun/dosyanın/repo içeriğinin kendisi bu kayda dahil değildir, yukarıdaki maddede anlatıldığı gibi hiçbir zaman sunucumuzda tutulmaz.

Bu özelliği Ayarlar sayfasından istediğin zaman kapatabilirsin; kapattığında yeni analizler kaydedilmez. Geçmiş sayfasından tek tek kayıt da silebilirsin.

GitHub token'ı

Private repo analizi için girdiğin GitHub token'ı sadece o anki isteği yapmak için kullanılır, hiçbir yere kaydedilmez.

Çerezler

Sadece oturumunu açık tutmak için teknik amaçlı bir oturum çerezi kullanılır. Reklam/takip amaçlı çerez yok.

Hesabını silmek istersen

API key'lerini Ayarlar sayfasından, analiz geçmişi kayıtlarını Geçmiş sayfasından kendin kaldırabilirsin. Hesabının tamamen silinmesini istersen aşağıdaki adresten bize yazman yeterli.
