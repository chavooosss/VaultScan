Google ile girişte ne alıyoruz

Sadece Google hesabından gelen ad, e-posta adresi ve profil fotoğrafı bağlantısı. Şifreni hiçbir zaman görmüyoruz — giriş tamamen Google üzerinden yapılır.

AI API key'lerin

VaultScan kendi AI key'ini kullanmaz — Claude/ChatGPT/Gemini için girdiğin key'ler senindir ve veritabanında şifrelenmiş olarak saklanır. Hiçbir ekranda veya API yanıtında düz metin olarak geri gösterilmez. İstediğin zaman Ayarlar sayfasından silebilirsin.

Analiz ettiğin kod ne olur

VaultScan, yapıştırdığın/yüklediğin kodu veya GitHub repo içeriğini kendi sunucusunda saklamaz. Kod, seçtiğin AI sağlayıcısına (Claude/ChatGPT/Gemini) senin kendi API key'inle iletilir, analiz orada yapılır ve sonuç tarayıcına gösterilir — VaultScan bu sonucu da kaydetmez.

Bu nedenle gerçekten gizli/hassas kod paylaşıyorsan, seçtiğin AI sağlayıcının kendi gizlilik politikasını da göz önünde bulundur — o aşamada veri o sağlayıcının sunucularında işlenir.

GitHub token'ı

Private repo analizi için girdiğin GitHub token'ı sadece o anki isteği yapmak için kullanılır, hiçbir yere kaydedilmez.
